doit gérer:
- jeux (nom, description, prix, categorie, auteur)
- categories de jeux
- auteurs 
- avis
- ...

fonctionnalités:
- catalogue
- avis
- jeux bien notés
- jeux beaucoup commentés
- /jeu ou /categorie
